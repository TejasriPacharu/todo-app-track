# Todo App (Action Items Manager)

A simple Todo (Action Items) application built to manage daily tasks with basic CRUD functionality.  
This project focuses on clean schema design, well-defined APIs, and a minimal UI as a foundation for future extensions.

---

## Features

- Create action items (todos)
- View all action items
- Update action item status (`todo`, `doing`, `done`)
- Delete action items
- Optional deadline for each action item
- Simple and clean MVP implementation

---

## Tech Stack

- **Frontend:** Next.js (React)
- **Backend:** Next.js API Routes
- **Language:** TypeScript
- **State Management:** In-memory store (Map)
- **Storage:** In-memory (MVP)
